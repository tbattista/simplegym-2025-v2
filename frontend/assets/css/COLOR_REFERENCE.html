Usage Examples

CSS Variables:
color: var(--gs-primary);
background: rgba(var(--gs-success-rgb), 0.1);
border-color: var(--gs-accent);

Pre-computed Opacity Variants:
background: var(--gs-primary-5); /* 5% opacity */
background: var(--gs-success-10); /* 10% opacity */
border: 1px solid var(--gs-danger-25); /* 25% opacity */
