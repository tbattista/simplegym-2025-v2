# Quick Notes Popover - Architecture & Implementation Plan

## Vision

A reusable, modular Quick Notes system that provides inline note-taking functionality across the workout mode and beyond. Starting with weight direction notes, but designed for extensibility.

---

## Design Concept

### Trigger Button
```
📝 ← Empty note icon (no notes set)
📝★ or 🔵📝 ← Filled/colored icon (has notes)
```

### Popover Content
```
┌─────────────────────────────┐
│  Quick Notes                │ ← Header (configurable)
├─────────────────────────────┤
│                             │
│  [Decrease] [Increase]      │ ← Quick action buttons
│                             │
│  ─────────── or ──────────  │
│                             │
│  [📝 Add text note...]      │ ← Future: text notes
│                             │
└─────────────────────────────┘
```

### Behaviors
- **Click trigger** → Popover opens
- **Click action button** → Action fires, popover closes
- **Click outside** → Popover closes
- **Icon state changes** when notes are set

## API Design

### QuickNotesPopover Class

```javascript
/**
 * QuickNotesPopover - Reusable quick notes popover component
 * 
 * Usage:
 * const popover = new QuickNotesPopover(triggerElement, {
 *     type: 'weight-direction',      // Preset type or 'custom'
 *     entityId: 'Bench Press',       // What this note is for
 *     onAction: (action, data) => {} // Callback when action taken
 * });
 */
class QuickNotesPopover {
    constructor(triggerElement, options) {
        this.trigger = triggerElement;
        this.options = {
            type: 'custom',           // 'weight-direction' | 'exercise-note' | 'custom'
            title: 'Quick Notes',
            entityId: null,
            entityType: 'exercise',   // 'exercise' | 'set' | 'workout'
            placement: 'bottom',      // Bootstrap popover placement
            actions: [],              // Custom actions array
            showTextInput: false,     // Whether to show text input
            textInputPlaceholder: 'Add a note...',
            currentValue: null,       // Current selected value
            onAction: null,           // Callback: (action, data) => {}
            onTextNote: null,         // Callback: (text) => {}
            onClose: null,            // Callback when popover closes
            ...options
        };
        
        this.init();
    }
    
    // Initialize the popover
    init() {}
    
    // Show the popover
    show() {}
    
    // Hide the popover
    hide() {}
    
    // Update the trigger icon based on state
    updateTriggerState(hasValue) {}
    
    // Set the current value
    setValue(value) {}
    
    // Get the current value
    getValue() {}
    
    // Destroy the popover
    destroy() {}
}
```

### workout-mode-controller.js Changes

```javascript
/**
 * Show quick notes popover for weight direction
 * @param {HTMLElement} trigger - The trigger button element
 */
showQuickNotes(trigger) {
    const exerciseName = trigger.getAttribute('data-exercise-name');
    const noteType = trigger.getAttribute('data-note-type');
    const currentValue = trigger.getAttribute('data-current-value');
    
    // Create and show the popover
    const popover = new QuickNotesPopover(trigger, {
        type: noteType,
        entityId: exerciseName,
        currentValue: currentValue || null,
        onAction: (action, data) => {
            this.handleQuickNoteAction(exerciseName, action, data);
        }
    });
    
    popover.show();
}

/**
 * Handle quick note action
 * @param {string} exerciseName - Exercise name
 * @param {string} action - Action taken (e.g., 'up', 'down')
 * @param {Object} data - Additional data
 */
handleQuickNoteAction(exerciseName, action, data) {
    if (data.noteType === 'weight-direction') {
        // Get current direction to check if toggling off
        const currentDirection = this.sessionService.getWeightDirection(exerciseName);
        const newDirection = (currentDirection === action) ? null : action;
        
        // Update session service
        this.sessionService.setWeightDirection(exerciseName, newDirection);
        
        // Update trigger button state
        this.updateQuickNoteTrigger(exerciseName, newDirection);
        
        // Show toast
        if (window.showAlert && newDirection) {
            const messages = {
                'up': 'Increase weight next session ⬆️',
                'down': 'Decrease weight next session ⬇️'
            };
            window.showAlert(messages[newDirection], 'success');
        }
        
        // Auto-save
        this.autoSave(null);
    }
}

/**
 * Update quick note trigger button state
 * @param {string} exerciseName - Exercise name
 * @param {*} value - Current value
 */
updateQuickNoteTrigger(exerciseName, value) {
    const trigger = document.querySelector(
        `.quick-notes-trigger[data-exercise-name="${exerciseName}"]`
    );
    
    if (!trigger) return;
    
    const icon = trigger.querySelector('i');
    
    if (value) {
        trigger.classList.add('has-note');
        trigger.setAttribute('data-current-value', value);
        if (icon) {
            icon.classList.remove('bx-note');
            icon.classList.add('bxs-note');
        }
    } else {
        trigger.classList.remove('has-note');
        trigger.setAttribute('data-current-value', '');
        if (icon) {
            icon.classList.remove('bxs-note');
            icon.classList.add('bx-note');
        }
    }
}
```

## Summary

This architecture provides:

1. **Single trigger button** - Clean, minimal UI
2. **Popover with actions** - Contextual quick notes
3. **Visual feedback** - Icon changes when notes are set
4. **Close on action** - Quick interaction, no extra clicks
5. **Reusable component** - Same pattern for different note types
6. **Future extensibility** - Text notes, performance ratings, etc.
7. **Saves to history** - Notes persist with workout data

The component is designed to be the foundation for a comprehensive notes system across the workout mode.
