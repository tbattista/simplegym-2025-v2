Authentication Flow

Users can sign in with email/password or Google OAuth. Anonymous users can still use core features with local storage.

Sign In / Sign Up Flow

flowchart LR
    A[Landing Page] --> B{Has Account?}
    B -->|Yes| C[Click Sign In]
    B -->|No| D[Click Sign Up]
    C --> E[Auth Modal - Sign In Tab]
    D --> F[Auth Modal - Sign Up Tab]
    E --> G{Method}
    F --> G
    G -->|Email| H[Enter Credentials]
    G -->|Google| I[Google OAuth]
    H --> J[Authenticated Dashboard]
    I --> J
    A -->|Skip| K[Use Anonymously]
    K --> L[Local Storage Mode]

Authentication Steps

1

Access Auth Modal

Click the user icon in the top-right navbar to open the authentication modal

2

Choose Method

Select Sign In or Sign Up tab, then choose Email/Password or Google OAuth

3

Complete Authentication

Enter credentials and submit. On success, you're redirected to the dashboard
